# 🏘️ Городок

Проксимити-чат с видом сверху: поле, рисованные стикмены, серверы-городки.
Слышно только тех, кто рядом. Подойди — услышишь, отойди — тишина.

## Запуск

```bash
npm install
node server.js
# открыть http://localhost:3000
```

## Что умеет

- **Без логина** — зашёл, придумал имя (или нет), болтаешь.
- **Городки (комнаты)** — любой текст в поле комнаты создаёт новый городок.
- **📍 Городок рядом** — комната по геолокации (геохэш ~1 км): соседи по дому,
  парку или району. «У кого есть соль?», «не шумите пожалуйста» и т.п.
- **Проксимити-чат** — сервер отдаёт сообщения только игрокам в радиусе
  слышимости; пузыри тают с расстоянием. Пунктирный круг — твой радиус.
- **Настраиваемый радиус** — сам выбираешь, на сколько метров тебя слышно
  (1–10 м, ползунок 📢 на входе и в верхней панели). Хочешь шептаться —
  ставь 1–2 м, хочешь вещать на всю площадь — 10 м.
- **Статусы** — текст над головой («блондинка в синем, ищу с кем покатать»),
  кнопка «💭 статус».
- **Тема беседы группы** — когда 2+ человека стоят рядом и болтают, над группой
  появляется общий ярлык с темой. С ключом `ANTHROPIC_API_KEY` тему формулирует
  Claude (модель настраивается через `ANTHROPIC_MODEL`, по умолчанию
  `claude-opus-4-8`); без ключа — извлечение ключевых слов.
- **🎤 Голос** — WebRTC-mesh, громкость собеседника падает с расстоянием.
  Работает на localhost или по https.
- **Мобильные** — виртуальный джойстик на тач-экранах.

## Управление

WASD / стрелки — ходить, Enter — чат, джойстик на телефоне.

## Деплой (бесплатно, с https)

Микрофон и геолокация в браузерах работают только по **https**, поэтому для
теста с друзьями нужен хостинг с https. GitHub Pages не подходит (статика без
WebSocket), зато подходит Render:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/glalker/gorodok)

1. Нажми кнопку выше (или: [render.com](https://render.com) → **New →
   Blueprint** → репозиторий `glalker/gorodok`). Конфиг подхватится из
   `render.yaml` автоматически.
2. Через пару минут получишь ссылку вида `https://gorodok.onrender.com` —
   кидай друзьям. Голос и «городок рядом» (геолокация) заработают.

На бесплатном тарифе Render сервер засыпает после 15 минут тишины — первый
заход после паузы грузится ~30 секунд, дальше летает.

## ИИ-темы бесед

```bash
ANTHROPIC_API_KEY=sk-ant-... node server.js
```
(на Render: добавь `ANTHROPIC_API_KEY` в Environment Variables сервиса)
